# reflect skill — local config

Copy this file to `config.md` (gitignored) and fill in your paths.

```
events_path: /path/to/your/event/store/events/
templates_path: /path/to/your/event/store/templates/
claude_md_path: /path/to/your/event/store/CLAUDE.md
```

## Example (Google Drive)

```
events_path: /Users/yourname/Library/CloudStorage/GoogleDrive-you@email.com/My Drive/Claude/Self-Learning/events/
templates_path: /Users/yourname/Library/CloudStorage/GoogleDrive-you@email.com/My Drive/Claude/Self-Learning/templates/
claude_md_path: /Users/yourname/Library/CloudStorage/GoogleDrive-you@email.com/My Drive/Claude/Self-Learning/CLAUDE.md
```

## Example (local folder)

```
events_path: ~/Documents/claude-learning/events/
templates_path: ~/Documents/claude-learning/templates/
claude_md_path: ~/Documents/claude-learning/CLAUDE.md
```
